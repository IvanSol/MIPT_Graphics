### MIPT Computer Graphics'17 assignments by Ivan Solomatin, 274, DCAM, MIPT.
#### Assignment 1. PostScript

В файлах ex*.eps - упражнения по отдельности.
В файле solution.eps - все упражнения, собранные в один файл.

Процедура рисования во всех упражнениях оформлялась в виде функции с изменяемыми параметрами (например, для сектора - x, y центра, внешний и внутренний радиусы, радиальный угол начала и конца сектора)

Репозиторий так же содержит .jpg и .png файлы с решением.
Артефакты jpeg проявляются только на низких разрешениях.
Размер .png файлов меньше, чем .jpg. Это ожидаемо, поскольку изображение содержит малое количество цветов (по большей части состоит из однотонных участков), а PNG хорошо сжимает подобные изображения.
